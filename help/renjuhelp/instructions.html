      ...
    1. 校准图片上面的坐标
        2. ...
      1. 设置行数，列数。
        2. 根据棋盘图片可视区域的行数，列数设置。
      2. 校准棋盘坐标
        3. 长按棋盘，触发拖动操作。使虚线框对齐棋盘，虚线圆圈尽可能对准过棋盘棋子。 PS：只要对准棋盘两个对角，就可以快速对齐棋盘。点击虚线框边角，可以微调整。有的棋盘图片 棋子不是正对棋盘交叉线，以虚线圆圈对正棋子为准。
    2. 自动识别棋子
        3. ...
      1. 点击自动识别
        2. PS：自动识别原理 是通过计算棋子坐标附近的 平均颜色来实现的。所以前面校准棋盘很重要，如果颜色太过丰富的棋盘 是很难识别成功的。
      2. 验证棋子是否准确
        3. 自动识别的棋子可能会有误差，这个时候就可以手动编辑一下棋子
      3. ●：添加，删除黑子
      4. ○：添加，删除白子
    3. 摆入棋子
      4. 如果不是标准的15路棋盘，需要定位天元点。长按(或鼠标右键点击)图片上面的天元点 进行定位。
